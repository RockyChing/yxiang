# 网格策略之两种优化方案

### 一、以低/高于格子的价格买/卖
   
如下图，**全天价格都在买入线的下方，那么就没必要一定用买入线价格买入**——能用更低的价格买入，何乐不为呢！   
   
全天价格都在买入线的上方，同理。   

原则就是，根据情况灵活变通。但尽量保证，不要因此而错过买/卖点。   
   
![yxtz_250401_01](..\Images\yxtz_250401_01.jpg)

### 二、结合趋势
   
下跌趋势，那么就没必要每个格子都买入——略过去的格子份额，在下面的格子一起买入。   
   
如下图，买点④的买入份额 = 格子① + ② + ③ + ④份额。   
   
上涨趋势，同理。
